
# Tunnel-Gott-Erkenner

## about

track seismic activity and calculate origin of burst


## raspberry

- client for raspberry that saves data in files.
- script that copies files to local server

- sensors used:

  - ADS1115 and GY-61
  - MMA7455L or MPU6050
  - Adafruit Ultimate GPS Breakout
